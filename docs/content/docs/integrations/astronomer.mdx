The Astronomer integration provides access to manage apache airflow dags and deployments on astronomer through the Integrate MCP server.

## Installation

```typescript
import { astronomerIntegration } from "integrate-sdk/server";
```

## Setup

Add Astronomer to your server configuration. This integration can read credentials from environment variables automatically (`ASTRO_API_TOKEN`, `ASTRONOMER_API_TOKEN`):

```typescript
import { createMCPServer, astronomerIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    astronomerIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

const result = await client.astronomer.getSelf({
  // Add the parameters required by this tool
});

console.log(result);
```
