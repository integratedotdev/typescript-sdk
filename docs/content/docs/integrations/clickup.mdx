The ClickUp integration provides access to manage clickup tasks, lists, spaces, and workspaces through the Integrate MCP server.

## Installation

```typescript
import { clickupIntegration } from "integrate-sdk/server";
```

## Setup

Add ClickUp to your server configuration. The integration automatically reads OAuth credentials from your environment (`CLICKUP_CLIENT_ID`, `CLICKUP_CLIENT_SECRET`):

```typescript
import { createMCPServer, clickupIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    clickupIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("clickup");
const result = await client.clickup.getAuthorizedUser({
  // Add the parameters required by this tool
});

console.log(result);
```
