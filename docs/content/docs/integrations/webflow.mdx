The Webflow integration provides access to manage webflow sites, collections, and cms content through the Integrate MCP server.

## Installation

```typescript
import { webflowIntegration } from "integrate-sdk/server";
```

## Setup

Add Webflow to your server configuration. The integration automatically reads OAuth credentials from your environment (`WEBFLOW_CLIENT_ID`, `WEBFLOW_CLIENT_SECRET`):

```typescript
import { createMCPServer, webflowIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    webflowIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("webflow");
const result = await client.webflow.tokenIntrospect({
  // Add the parameters required by this tool
});

console.log(result);
```
