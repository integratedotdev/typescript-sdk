The Xero integration provides access to manage xero invoices, contacts, and accounting data through the Integrate MCP server.

## Installation

```typescript
import { xeroIntegration } from "integrate-sdk/server";
```

## Setup

Add Xero to your server configuration. The integration automatically reads OAuth credentials from your environment (`XERO_CLIENT_ID`, `XERO_CLIENT_SECRET`):

```typescript
import { createMCPServer, xeroIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    xeroIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("xero");
const result = await client.xero.listConnections({
  // Add the parameters required by this tool
});

console.log(result);
```
