The Redis Cloud integration provides access to manage redis cloud subscriptions, databases, async tasks, and audit logs through the Integrate MCP server.

## Installation

```typescript
import { encodeRedisCloudBearerToken } from "integrate-sdk/server";
```

## Setup

Add Redis Cloud to your server configuration. This integration can read credentials from environment variables automatically (`REDIS_CLOUD_ACCOUNT_KEY`, `REDIS_CLOUD_SECRET_KEY`):

```typescript
import { createMCPServer, encodeRedisCloudBearerToken } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    encodeRedisCloudBearerToken(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

const result = await client.redis.listSubscriptions({
  // Add the parameters required by this tool
});

console.log(result);
```
