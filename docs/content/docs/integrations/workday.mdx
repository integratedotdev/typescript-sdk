The Workday integration provides access to read workday hr data including workers, organizations, and pay through the Integrate MCP server.

## Installation

```typescript
import { workdayIntegration } from "integrate-sdk/server";
```

## Setup

Add Workday to your server configuration. The integration automatically reads OAuth credentials from your environment (`WORKDAY_CLIENT_ID`, `WORKDAY_CLIENT_SECRET`, `WORKDAY_TENANT_DESCRIPTOR`):

```typescript
import { createMCPServer, workdayIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    workdayIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("workday");
const result = await client.workday.listWorkers({
  // Add the parameters required by this tool
});

console.log(result);
```
