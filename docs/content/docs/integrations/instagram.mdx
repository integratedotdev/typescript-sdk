The Instagram integration provides access to instagram graph api — media, comments, insights, stories, and publishing through the Integrate MCP server.

## Installation

```typescript
import { instagramIntegration } from "integrate-sdk/server";
```

## Setup

Add Instagram to your server configuration. The integration automatically reads OAuth credentials from your environment (`INSTAGRAM_CLIENT_ID`, `INSTAGRAM_CLIENT_SECRET`):

```typescript
import { createMCPServer, instagramIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    instagramIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("instagram");
const result = await client.instagram.listPages({
  // Add the parameters required by this tool
});

console.log(result);
```
