The Zapier integration provides access to trigger and manage zapier automation workflows through the Integrate MCP server.

## Installation

```typescript
import { zapierIntegration } from "integrate-sdk/server";
```

## Setup

Add Zapier to your server configuration. The integration automatically reads OAuth credentials from your environment (`ZAPIER_CLIENT_ID`, `ZAPIER_CLIENT_SECRET`):

```typescript
import { createMCPServer, zapierIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    zapierIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("zapier");
const result = await client.zapier.getProfile({
  // Add the parameters required by this tool
});

console.log(result);
```
