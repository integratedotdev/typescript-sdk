The GitLab integration provides access to manage gitlab repositories, issues, and merge requests through the Integrate MCP server.

## Installation

```typescript
import { gitlabIntegration } from "integrate-sdk/server";
```

## Setup

Add GitLab to your server configuration. The integration automatically reads OAuth credentials from your environment (`GITLAB_CLIENT_ID`, `GITLAB_CLIENT_SECRET`):

```typescript
import { createMCPServer, gitlabIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    gitlabIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("gitlab");
const result = await client.gitlab.listProjects({
  // Add the parameters required by this tool
});

console.log(result);
```

## Environment Variables

- `GITLAB_CLIENT_ID`
- `GITLAB_CLIENT_SECRET`

## Default Scopes

These defaults are applied unless you override `scopes` in the integration config:

- `read_api`
- `write_api`
- `read_user`
