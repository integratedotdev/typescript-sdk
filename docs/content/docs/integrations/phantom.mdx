The Phantom integration provides access to build phantom mobile browse universal links and deeplink provider documentation (solana) through the Integrate MCP server.

## Installation

```typescript
import { buildPhantomBrowseDeeplink } from "integrate-sdk/server";
```

## Setup

Add Phantom to your server configuration. The integration automatically reads OAuth credentials from your environment:

```typescript
import { createMCPServer, buildPhantomBrowseDeeplink } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    buildPhantomBrowseDeeplink(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("phantom");
const result = await client.phantom.buildBrowseDeeplink({
  // Add the parameters required by this tool
});

console.log(result);
```
