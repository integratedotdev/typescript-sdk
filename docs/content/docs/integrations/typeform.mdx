The Typeform integration provides access to create and manage typeform surveys and responses through the Integrate MCP server.

## Installation

```typescript
import { typeformIntegration } from "integrate-sdk/server";
```

## Setup

Add Typeform to your server configuration. The integration automatically reads OAuth credentials from your environment (`TYPEFORM_CLIENT_ID`, `TYPEFORM_CLIENT_SECRET`):

```typescript
import { createMCPServer, typeformIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    typeformIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("typeform");
const result = await client.typeform.getMe({
  // Add the parameters required by this tool
});

console.log(result);
```

## Notes

- Category: Business
