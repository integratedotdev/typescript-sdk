The tldraw integration provides access to create and manage tldraw collaborative whiteboards through the Integrate MCP server.

## Installation

```typescript
import { tldrawIntegration } from "integrate-sdk/server";
```

## Setup

Add tldraw to your server configuration. The integration automatically reads OAuth credentials from your environment (`TLDRAW_API_KEY`):

```typescript
import { createMCPServer, tldrawIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    tldrawIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("tldraw");
const result = await client.tldraw.unfurlUrl({
  // Add the parameters required by this tool
});

console.log(result);
```
