The Upstash integration provides access to use upstash redis rest, qstash messaging, and related http apis through the Integrate MCP server.

## Installation

```typescript
import { upstashIntegration } from "integrate-sdk/server";
```

## Setup

Add Upstash to your server configuration. This integration can read credentials from environment variables automatically (`UPSTASH_REDIS_REST_URL`, `UPSTASH_REDIS_REST_TOKEN`, `QSTASH_TOKEN`):

```typescript
import { createMCPServer, upstashIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    upstashIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

const result = await client.upstash.redisRun({
  // Add the parameters required by this tool
});

console.log(result);
```

- `upstash_redis_set`

- `upstash_qstash_publish`
