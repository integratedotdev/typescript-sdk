The Dropbox Paper integration provides access to create, update, and export dropbox paper documents through the Integrate MCP server.

## Installation

```typescript
import { paperIntegration } from "integrate-sdk/server";
```

## Setup

Add Dropbox Paper to your server configuration. The integration automatically reads OAuth credentials from your environment (`PAPER_CLIENT_ID`, `DROPBOX_CLIENT_ID`, `PAPER_CLIENT_SECRET`, `DROPBOX_CLIENT_SECRET`):

```typescript
import { createMCPServer, paperIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    paperIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("paper");
const result = await client.paper.createDoc({
  // Add the parameters required by this tool
});

console.log(result);
```
