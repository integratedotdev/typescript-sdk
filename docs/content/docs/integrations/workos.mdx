The WorkOS integration provides access to manage workos organizations, users, directory sync, and authkit through the Integrate MCP server.

## Installation

```typescript
import { workosIntegration } from "integrate-sdk/server";
```

## Setup

Add WorkOS to your server configuration. This integration can read credentials from environment variables automatically (`WORKOS_API_KEY`):

```typescript
import { createMCPServer, workosIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    workosIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

const result = await client.workos.listOrganizations({
  // Add the parameters required by this tool
});

console.log(result);
```
