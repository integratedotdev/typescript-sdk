The eToro integration provides access to read etoro portfolio data and market instruments through the Integrate MCP server.

## Installation

```typescript
import { etoroIntegration } from "integrate-sdk/server";
```

## Setup

Add eToro to your server configuration. This integration can read credentials from environment variables automatically (`ETORO_PUBLIC_API_KEY`, `ETORO_USER_KEY`):

```typescript
import { createMCPServer, etoroIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    etoroIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

const result = await client.etoro.getIdentity({
  // Add the parameters required by this tool
});

console.log(result);
```
