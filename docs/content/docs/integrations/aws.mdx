The Amazon Web Services integration provides access to inspect aws accounts and resources with sigv4 (control-plane apis) through the Integrate MCP server.

## Installation

```typescript
import { encodeAwsCredentialsPayload } from "integrate-sdk/server";
```

## Setup

Add Amazon Web Services to your server configuration. This integration can read credentials from environment variables automatically (`AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY`, `AWS_SESSION_TOKEN`, `AWS_REGION`, `AWS_DEFAULT_REGION`):

```typescript
import { createMCPServer, encodeAwsCredentialsPayload } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    encodeAwsCredentialsPayload(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

const result = await client.aws.stsGetCallerIdentity({
  // Add the parameters required by this tool
});

console.log(result);
```

- `aws_sts_get_caller_identity`
- `aws_ec2_describe_regions`
- `aws_ec2_describe_instances`
- `aws_s3_list_buckets`
- `aws_lambda_list_functions`
- `aws_cloudformation_list_stacks`
- `aws_iam_list_account_aliases`
