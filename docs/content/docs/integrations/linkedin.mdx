The LinkedIn integration provides access to post updates and manage linkedin profile and company pages through the Integrate MCP server.

## Installation

```typescript
import { linkedinIntegration } from "integrate-sdk/server";
```

## Setup

Add LinkedIn to your server configuration. The integration automatically reads OAuth credentials from your environment (`LINKEDIN_CLIENT_ID`, `LINKEDIN_CLIENT_SECRET`):

```typescript
import { createMCPServer, linkedinIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    linkedinIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("linkedin");
const result = await client.linkedin.getUserinfo({
  // Add the parameters required by this tool
});

console.log(result);
```
