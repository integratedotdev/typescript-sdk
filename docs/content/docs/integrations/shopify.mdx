The Shopify integration provides access to manage shopify products, orders, customers, and inventory through the Integrate MCP server.

## Installation

```typescript
import { shopifyIntegration } from "integrate-sdk/server";
```

## Setup

Add Shopify to your server configuration. The integration automatically reads OAuth credentials from your environment (`SHOPIFY_CLIENT_ID`, `SHOPIFY_CLIENT_SECRET`, `SHOPIFY_SHOP`):

```typescript
import { createMCPServer, shopifyIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    shopifyIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("shopify");
const result = await client.shopify.adminGraphql({
  // Add the parameters required by this tool
});

console.log(result);
```
