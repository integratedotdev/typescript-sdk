The Google Chat integration provides access to send messages and manage google chat spaces and memberships through the Integrate MCP server.

## Installation

```typescript
import { gchatIntegration } from "integrate-sdk/server";
```

## Setup

Add Google Chat to your server configuration. The integration automatically reads OAuth credentials from your environment (`GCHAT_CLIENT_ID`, `GCHAT_CLIENT_SECRET`):

```typescript
import { createMCPServer, gchatIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    gchatIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("gchat");
const result = await client.gchat.deleteMessage({
  // Add the parameters required by this tool
});

console.log(result);
```
