The Google Contacts integration provides access to list, search, create, update, and delete google contacts via the people api through the Integrate MCP server.

## Installation

```typescript
import { gcontactsIntegration } from "integrate-sdk/server";
```

## Setup

Add Google Contacts to your server configuration. The integration automatically reads OAuth credentials from your environment (`GCONTACTS_CLIENT_ID`, `GCONTACTS_CLIENT_SECRET`):

```typescript
import { createMCPServer, gcontactsIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    gcontactsIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("gcontacts");
const result = await client.gcontacts.batchGetContacts({
  // Add the parameters required by this tool
});

console.log(result);
```
