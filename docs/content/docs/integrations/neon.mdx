The Neon integration provides access to manage neon postgres projects, branches, and databases through the Integrate MCP server.

## Installation

```typescript
import { neonIntegration } from "integrate-sdk/server";
```

## Setup

Add Neon to your server configuration. This integration can read credentials from environment variables automatically (`NEON_API_KEY`):

```typescript
import { createMCPServer, neonIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    neonIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

const result = await client.neon.listApiKeys({
  // Add the parameters required by this tool
});

console.log(result);
```
