The Google Tasks integration provides access to manage google tasks lists and to-dos through the Integrate MCP server.

## Installation

```typescript
import { gtasksIntegration } from "integrate-sdk/server";
```

## Setup

Add Google Tasks to your server configuration. The integration automatically reads OAuth credentials from your environment (`GTASKS_CLIENT_ID`, `GTASKS_CLIENT_SECRET`):

```typescript
import { createMCPServer, gtasksIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    gtasksIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("gtasks");
const result = await client.gtasks.clearCompleted({
  // Add the parameters required by this tool
});

console.log(result);
```
