The PostHog integration provides access to read posthog organizations, projects, insights, and feature flags through the Integrate MCP server.

## Installation

```typescript
import { posthogIntegration } from "integrate-sdk/server";
```

## Setup

Add PostHog to your server configuration. The integration automatically reads OAuth credentials from your environment (`POSTHOG_CLIENT_ID`, `POSTHOG_CLIENT_SECRET`):

```typescript
import { createMCPServer, posthogIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    posthogIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("posthog");
const result = await client.posthog.getCurrentUser({
  // Add the parameters required by this tool
});

console.log(result);
```
