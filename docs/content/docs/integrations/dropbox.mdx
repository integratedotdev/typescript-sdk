The Dropbox integration provides access to manage dropbox files, folders, and sharing through the Integrate MCP server.

## Installation

```typescript
import { dropboxIntegration } from "integrate-sdk/server";
```

## Setup

Add Dropbox to your server configuration. The integration automatically reads OAuth credentials from your environment (`DROPBOX_CLIENT_ID`, `DROPBOX_CLIENT_SECRET`):

```typescript
import { createMCPServer, dropboxIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    dropboxIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("dropbox");
const result = await client.dropbox.getCurrentAccount({
  // Add the parameters required by this tool
});

console.log(result);
```
