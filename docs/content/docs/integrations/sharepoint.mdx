The SharePoint integration provides access to manage sharepoint sites, lists, documents, and permissions through the Integrate MCP server.

## Installation

```typescript
import { sharepointIntegration } from "integrate-sdk/server";
```

## Setup

Add SharePoint to your server configuration. The integration automatically reads OAuth credentials from your environment (`SHAREPOINT_CLIENT_ID`, `SHAREPOINT_CLIENT_SECRET`):

```typescript
import { createMCPServer, sharepointIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    sharepointIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("sharepoint");
const result = await client.sharepoint.createFolder({
  // Add the parameters required by this tool
});

console.log(result);
```
