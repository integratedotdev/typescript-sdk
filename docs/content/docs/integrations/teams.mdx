The Microsoft Teams integration provides access to send messages and manage microsoft teams channels and meetings through the Integrate MCP server.

## Installation

```typescript
import { teamsIntegration } from "integrate-sdk/server";
```

## Setup

Add Microsoft Teams to your server configuration. The integration automatically reads OAuth credentials from your environment (`TEAMS_CLIENT_ID`, `TEAMS_CLIENT_SECRET`):

```typescript
import { createMCPServer, teamsIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    teamsIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("teams");
const result = await client.teams.getChannel({
  // Add the parameters required by this tool
});

console.log(result);
```
