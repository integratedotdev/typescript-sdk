The Convex integration provides access to run convex functions, manage tables, and query your app backend through the Integrate MCP server.

## Installation

```typescript
import { convexIntegration } from "integrate-sdk/server";
```

## Setup

Add Convex to your server configuration. This integration can read credentials from environment variables automatically (`CONVEX_ACCESS_TOKEN`):

```typescript
import { createMCPServer, convexIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    convexIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

const result = await client.convex.managementTokenDetails({
  // Add the parameters required by this tool
});

console.log(result);
```
