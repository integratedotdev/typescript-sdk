The Zoom integration provides access to manage zoom user profile and meetings through the Integrate MCP server.

## Installation

```typescript
import { zoomIntegration } from "integrate-sdk/server";
```

## Setup

Add Zoom to your server configuration. The integration automatically reads OAuth credentials from your environment (`ZOOM_CLIENT_ID`, `ZOOM_CLIENT_SECRET`):

```typescript
import { createMCPServer, zoomIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    zoomIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("zoom");
const result = await client.zoom.getUser({
  // Add the parameters required by this tool
});

console.log(result);
```
