The Better Stack integration provides access to monitor uptime, logs, and incident alerts with betterstack through the Integrate MCP server.

## Installation

```typescript
import { betterstackIntegration } from "integrate-sdk/server";
```

## Setup

Add Better Stack to your server configuration. This integration can read credentials from environment variables automatically (`BETTERSTACK_API_KEY`):

```typescript
import { createMCPServer, betterstackIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    betterstackIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

const result = await client.betterstack.listSources({
  // Add the parameters required by this tool
});

console.log(result);
```
