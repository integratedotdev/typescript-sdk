The Attio integration provides access to manage attio people, companies, tasks, and crm records through the Integrate MCP server.

## Installation

```typescript
import { attioIntegration } from "integrate-sdk/server";
```

## Setup

Add Attio to your server configuration. The integration automatically reads OAuth credentials from your environment (`ATTIO_CLIENT_ID`, `ATTIO_CLIENT_SECRET`):

```typescript
import { createMCPServer, attioIntegration } from "integrate-sdk/server";

export const { client: serverClient } = createMCPServer({
  apiKey: process.env.INTEGRATE_API_KEY,
  integrations: [
    attioIntegration(),
  ],
});
```

### Client-Side Usage

```typescript
import { client } from "integrate-sdk";

await client.authorize("attio");
const result = await client.attio.getSelf({
  // Add the parameters required by this tool
});

console.log(result);
```
